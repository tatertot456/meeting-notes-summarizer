Bayesian Belief Networks

▶

Bayesian Belief Networks (BBNs) allow class conditional independencies between subsets of variables — more flexible than Naïve Bayes which assumes full independence.

  • Represented as a directed acyclic graph (DAG)
  • Nodes = random variables, Links = dependencies
  • Each node has a Conditional Probability Table (CPT)
  • Captures causal relationships between variables

Known Structure, All Observable

Simply compute CPT entries from data.

Known Structure, Hidden Variables

Use gradient descent (greedy hill-climbing) to converge to local optimum.

Unknown Structure, All Observable

Search model space to reconstruct network topology.

Unknown Structure, Hidden Variables

No good algorithms known for this scenario.

Neural Networks & Backpropagation

▶

Neural networks learn by adjusting weights to minimize error between predicted and actual outputs. Training uses backpropagation — propagating error backwards from output to input layers.

Strengths

Tolerates noisy data, classifies untrained patterns, works well for continuous values, real-world success (handwriting, image recognition).

Weaknesses

Long training time, many empirical parameters (topology, learning rate), poor interpretability of learned weights.

Backpropagation steps:

  • Initialize weights to small random numbers
  • Propagate inputs forward through activation functions
  • Backpropagate error — update weights and biases
  • Terminate when error is below threshold or max epochs reached

More hidden layers = more complex decision boundaries. Neural networks perform nonlinear regression — with enough hidden units and training data, they can approximate any function.

Deep Learning & CNNs

▶

Convolutional Neural Networks (CNNs) revolutionized image classification. AlexNet dropped ImageNet error from 26% to 15% in 2012.

Convolution

Apply filters to detect features (edges, curves). Filter depth = number of filters. Stride = pixel step between filters.

ReLU

Rectified Linear Unit: output = max(0, input). Removes negatives from feature maps. Works better than tanh/sigmoid.

Pooling

Reduces spatial resolution of feature maps. Max pooling is most common. Reduces computation and overfitting.

Iterative Process

1st layers find shapes/edges. Deeper layers find object parts. Final layers fed into fully connected ANN for classification.

Object detection variants:

  • R-CNN: Selective search + CNN per region + SVM classification
  • Fast/Faster R-CNN: ROI pooling for performance improvement
  • Mask R-CNN: Full pixel-level object segmentation using FCN

Support Vector Machines (SVM)

▶

SVMs find the Maximum Margin Hyperplane (MMH) — the decision boundary furthest from the nearest training tuples of both classes.

Support Vectors

Training tuples that lie on the margin. These alone define the hyperplane — removing other tuples and retraining yields the same result.

Kernel Functions

Map data to higher dimensions to find a linear separator. Common kernels: polynomial, Gaussian RBF, sigmoid. Avoids computing in high-dimensional space.

vs. Neural Networks

SVM: deterministic, strong math foundation, batch learning. Neural Net: nondeterministic, incremental learning, complex functions via multilayer perceptron.

Multiclass SVM

One-vs-All: m classifiers for m classes. All-vs-All: m(m-1)/2 classifiers. AVA tends to be superior.

SVM effectiveness on high-dimensional data comes from complexity being characterized by the number of support vectors, not the data dimensionality.

Lazy Learners (k-NN & Case-Based Reasoning)

▶

Lazy learners store training data and wait until a test tuple arrives — less training time, more prediction time.

k-Nearest Neighbor

Classify by majority vote among k nearest neighbors (Euclidean distance). Distance-weighted variant gives closer neighbors more influence.

Case-Based Reasoning

Uses a database of symbolic problem solutions. Retrieves similar cases, possibly combines multiple, and adapts to new problems.

Curse of dimensionality: irrelevant attributes can dominate distance calculations in k-NN. Overcome via axis stretching or attribute elimination.

Lazy methods use many local linear functions forming an implicit global approximation — richer hypothesis space than eager learners that commit to one global model.

Semi-Supervised & Generative Methods

▶

Self-Training

Build classifier from labeled data, label the most confident unlabeled predictions, repeat. Risk: may reinforce errors.

Co-Training

Two classifiers on independent feature sets teach each other by exchanging most confident predictions.

GANs

Generative network creates candidates; discriminative network evaluates against real data. Generator tries to fool discriminator. Used for art, music, deepfakes.

Autoencoders

Compress input to latent representation then reconstruct. Used for compression, denoising, and feature extraction. Loss = MSE (continuous) or cross-entropy (binary).

Contrastive Learning (SimCLR, FireCLR): uses data augmentation (random crop, blur, flip, rotation, color change) to learn representations without labels.

Transfer Learning & Reinforcement Learning

▶

Transfer Learning

Apply knowledge from source tasks to a new target task. Useful when target data is scarce or distribution changes (e.g., spam filtering, web classification).

TrAdaBoost

Reduces weight of misclassified source tuples so they have less effect on subsequent classifiers. Focuses model on the target domain.

Reinforcement Learning

Agent learns by reward/penalty. Q-Learning builds a Q(state, action) table estimating expected reward. Balance exploration vs exploitation.

Deep Reinforcement Learning

Replaces Q-table with a neural network (weight matrices). Combines CNN feature extraction with RL reward-based learning.

Graph Neural Networks

▶

GNNs operate on graph-structured data where relationships between entities matter. Applications: social networks, drug discovery, recommender systems.

Message Passing

Nodes receive information from adjacent nodes — giving context from neighbors. Each phase of message passing = one layer. Implemented using adjacency matrix operations.

Embedding

Maps nodes into a numerical space preserving topology. More similar nodes end up closer in embedding space. Forward/backward propagation improves embeddings.

Node Classification

Predict properties of individual nodes (categorize users/items).

Link Prediction

Find missing links between nodes — core of recommender systems. Uses proximity of embeddings.
