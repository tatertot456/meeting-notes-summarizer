Symbols, Color Models & Visual Variables

▶

Map symbols communicate two types of information through visual variables:

Categorical Differences

Expressed by varying symbol SHAPE, line type, pattern, color hue, or font. Use distinct colors (red, blue, orange) — not shades of the same color.

Quantity Differences

Expressed by varying symbol SIZE, line thickness, or color shade/saturation. Use shades of one color for ordered quantities.

Color Guidelines:

  • Different hues (colors) → categories
  • Different shades of the same hue → quantities (low to high)
  • Two-color diverging scheme → data with a meaningful center (negative/positive, cold/warm)
  • Avoid rainbow color schemes for quantities — too many hues make maps harder to interpret

RGB Color Model: Based on mixtures of Red, Green, Blue light. Each primary ranges 0 (black) to 255 (bright). Over 16 million possible combinations (256³).

HSV Color Model:

  • Hue: the color itself on a 0–360° color wheel
  • Saturation: intensity/purity of the color (0–100%)
  • Value: brightness (0–100%)

HSV is more intuitive for cartographic decisions than RGB. To make a quantity map, keep Hue fixed and vary Saturation or Value. To make a category map, vary Hue while keeping Saturation and Value similar so no one category appears more important than another.

The distinction between interval and ratio matters for normalization. You can say "twice as many farms" (ratio) but not "twice as hot" in Fahrenheit (interval — because 0°F isn't true absence of heat). Both are mapped similarly in GIS but the choice of normalization and interpretation differs.

Map Types for Different Data

▶

Single Symbol Map

All features shown with the same symbol. Used for nominal data where each feature is unique. Labels added to identify individual features.

Unique Values Map

Each category gets a distinct symbol/color. Used for categorical and ordinal data. Don't use for quantities — implies categories not magnitudes.

Graduated Color (Choropleth)

Colors vary by data class — darker = more. Used for interval/ratio data. AVOID rainbow schemes; use sequential single-hue or diverging two-hue.

Graduated Symbol

Symbol SIZE varies by data value. Better than graduated color for avoiding visual MAUP — large polygons don't dominate. Used for quantities.

Proportional Symbol

Unclassed — symbol size increases in exact proportion to value. No artificial grouping into classes.

Dot Density Map

Random dots placed within polygons based on a ratio (e.g., 1 dot = 1,000 farms). Unclassed — avoids artificial grouping. Shows distribution within polygons.

Classed maps (choropleth, graduated symbol) place features into ranges. The classification method significantly impacts map appearance and interpretation — the same data can tell very different stories depending on how classes are defined. Unclassed maps (proportional symbol, dot density) avoid this subjectivity but can be harder to read for precise values.

Modifiable Areal Unit Problem (MAUP)

▶

The MAUP occurs when arbitrary aggregation units (states, counties, census tracts) distort the patterns being mapped. The map reflects the units more than the underlying data.

Statistical MAUP

Number of farms in a state is influenced by the STATE'S SIZE — large states look like they have more farms even if farm density is low. Number of vacant houses influenced by state population.

Visual MAUP

Large polygons dominate the map visually and attract more attention, even when their values are low. Mountain states stand out in farm density maps. Maine's high vacancy rate goes unnoticed nationally.

Solutions:

  • Normalize (divide) by a suitable field — farms per sq mile instead of total farms; fraction of vacant housing instead of count
  • Normalize by percentage — divide each value by the total (e.g., % of Congress from each state)
  • Use graduated symbol instead of graduated color — symbol size isn't tied to polygon size, so large polygons don't visually dominate
  • Use dot density maps — dots distributed within polygons show where density actually is

Always normalize ratio data before mapping counts in polygons of varying size. "Farms per square mile" tells a different (and more honest) story than "total farms." This is one of the most common mistakes in GIS mapping. When in doubt: normalize!

Raster Data Types & Display

▶

Discrete Rasters

Represent objects (roads, land use). Few distinct values. Adjacent cells often share values. Abrupt boundaries. Displayed with unique values or discrete color.

Continuous Rasters

Represent measurements everywhere (elevation, temperature). Millions of potential values. Gradual transitions. Displayed with classified or stretched methods.

Thematic Rasters

Contain map data quantities (geology, elevation). Can be discrete or continuous. Examples: classified elevation, continuous DEM.

Image Rasters

Satellite/aerial photo data — represents brightness. Single band: stretched display. Multi-band: RGB composite (bands assigned to R, G, B channels).

Display Methods:

MethodUsed ForHow It Works
Unique ValuesCategorical/ordinal rastersEach distinct value gets its own color. Like a unique values vector map.
ClassifiedInterval/ratio rastersValues grouped into classes, each with a color. Same classification methods as vector.
StretchedContinuous rasters, single-band imagesReallocates a portion of raster values to the 256 available display shades. Improves contrast. Types: min-max, standard deviation.
RGB CompositeMulti-band image rastersThree bands assigned to Red, Green, Blue display channels. Natural color or false color composites.
Indexed ColorSingle integer band with colormapEach integer value maps to a specific RGB color. Simple, consistent, storage-efficient. Used in USGS Digital Raster Graphics.

Data Classification Methods

▶

Classification divides continuous data into class ranges, each with its own symbol. The method significantly affects map appearance — the same data can look very different. Choice depends on the data distribution and map objective.

MethodHow It WorksBest For
Jenks Natural BreaksExploits natural gaps in the data distribution. Minimizes variance within classes, maximizes between classes.Unevenly distributed or skewed data. ArcGIS default — works well for most datasets.
Equal IntervalDivides the data range into equal-sized classes. User specifies number of classes.Uniformly distributed data. Easy to understand class ranges.
Defined IntervalUser specifies the class size (range). Number of classes determined automatically.When the interval has a meaningful value (e.g., every 10 years, every 100 people).
QuantileEqual NUMBER of features in each class (not equal range). Different features per class.Comparing relative rank. Produces visually balanced maps.
Standard DeviationClasses based on standard deviations above and below the mean.Showing how values compare to average. Diverging data around a mean.
Geometrical IntervalMultiplies each class range by a constant factor. Creates geometrically increasing intervals.Strongly skewed or logarithmically increasing data (population density, income).

Classification is a form of data exploration. Try multiple methods on the same dataset to see which best reveals the pattern you're trying to communicate — or to reveal what patterns exist. Quantile always produces a visually balanced map but can mislead: all classes appear equally important even if most data clusters at one end. Natural Breaks is the safest default for unknown distributions.

Layers, Layer Files & Styles

▶

Layers

Source data become layers when added to a map. Layers store display and label properties. The same source data can be visualized many different ways across different layers/maps — the source data is never modified.

Layer Files (.lyrx)

Save a layer's symbology settings to a file. Easy way to transfer identical symbology to multiple maps or projects. Useful for enforcing consistent symbols across an organization (e.g., arsenic monitoring sites always use the same symbol).

Styles

A collection of symbols stored together, often with a specific theme or purpose. Viewed and applied in the Symbology pane → Gallery tab. Organizations can create custom styles for consistent branding.
