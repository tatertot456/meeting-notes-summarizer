Data Objects & Attribute Types

▶

Data sets are made up of data objects (rows) described by attributes (columns). Attributes come in several types:

TypeDescriptionExample
NominalCategories with no orderHair color, zip code
BinaryTwo states onlyGender, COVID positive/negative
OrdinalOrdered but unknown magnitude between valuesGrades, military rank
IntervalEqual-sized units, no true zeroTemperature °C, calendar dates
RatioInherent zero pointTemperature K, length, monetary quantities

Discrete attributes have finite values (zip codes, professions). Continuous attributes have real number values (temperature, height, weight).

Basic Statistical Descriptions

▶

Understanding your data starts with measuring central tendency and dispersion:

Mean

Arithmetic average. Sensitive to outliers. Weighted and trimmed variations exist.

Median

Middle value. More robust than mean for skewed data.

Mode

Most frequent value. Data can be unimodal, bimodal, or trimodal.

Variance & Std Dev

Measures spread of data. Standard deviation is the square root of variance.

The five-number summary (Min, Q1, Median, Q3, Max) is displayed via a boxplot. Outliers are typically values beyond 1.5 × IQR from Q1 or Q3.

The normal distribution contains ~68% of data within ±1σ, ~95% within ±2σ, and ~99.7% within ±3σ of the mean.

Data Visualization

▶

Visualization helps gain insight, find patterns, and tell the story of your data to decision makers.

Histogram

Frequency distribution using bars. Area (not height) represents value when bins vary in width.

Quantile Plot

Shows each value paired with its percentile rank. Good for seeing distribution shape.

Q-Q Plot

Compares quantiles of two distributions. Reveals shifts between populations.

Scatter Plot

Each pair of values plotted as coordinates. Reveals clusters, outliers, and correlations.

Parallel Coordinates

N equidistant parallel axes for N attributes. Each data item is a polygonal line.

Chernoff Faces

Encodes variables as facial features. Leverages human ability to read faces.

Measuring Similarity & Dissimilarity

▶

Similarity is higher when objects are more alike (range 0–1). Dissimilarity (distance) is lower when objects are more alike.

Minkowski Distance

General distance formula. h=1: Manhattan. h=2: Euclidean. h=∞: Supremum (max difference).

Cosine Similarity

Used for text/document vectors. cos(d1,d2) = (d1·d2) / (||d1|| × ||d2||). Approaches 1 = more similar.

Jaccard Coefficient

Similarity for asymmetric binary variables. Used in set comparison.

Z-Score Standardization

Normalizes data: (x - μ) / σ. Makes attributes comparable regardless of scale.

For mixed attribute types, use a weighted formula combining nominal, binary, numeric, and ordinal distance measures.
