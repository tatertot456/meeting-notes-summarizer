Character Testing

▶

Requires #include <cctype>. These functions test a single character and return a nonzero (true) or zero (false) value.

FunctionReturns true if...
isalpha(ch)ch is a letter (A-Z or a-z)
isdigit(ch)ch is a digit (0-9)
isalnum(ch)ch is a letter or digit
isspace(ch)ch is whitespace (space, tab, newline)
isupper(ch)ch is an uppercase letter
islower(ch)ch is a lowercase letter
ispunct(ch)ch is a punctuation character
isprint(ch)ch is a printable character
#include <cctype>

char input = 'A';
if (isalpha(input))
    cout << "It's a letter\n";
if (isdigit(input))
    cout << "It's a digit\n";

Character Case Conversion

▶

Also requires #include <cctype>. These functions return the converted character — they do NOT modify the original.

char ch1 = 'H', ch2 = 'e', ch3 = '!';

// toupper: converts lowercase to uppercase, others unchanged
cout << toupper(ch1);  // 'H' (already upper, unchanged)
cout << toupper(ch2);  // 'E'
cout << toupper(ch3);  // '!' (not a letter, unchanged)

// tolower: converts uppercase to lowercase, others unchanged
cout << tolower(ch1);  // 'h'
cout << tolower(ch2);  // 'e' (already lower, unchanged)
cout << tolower(ch3);  // '!'

// Common use: normalize input for comparison
char answer;
cin >> answer;
if (toupper(answer) == 'Y')
    cout << "You said yes\n";

C-Strings

▶

A C-string is a sequence of characters stored in adjacent memory locations and terminated by a null character (\0). String literals like "Hi there!" are automatically null-terminated — "Hi there!" actually takes 10 bytes, not 9.

// Define a char array to hold a C-string
const int SIZE = 20;
char city[SIZE];         // can hold up to 19 chars + \0

// Initialize at definition
char name[] = "Missoula";  // automatically sized to 9 bytes (8 + \0)

// Input with cin >> (stops at whitespace)
cin >> city;

// Input with getline (reads whole line including spaces)
cin.getline(city, SIZE);   // reads up to SIZE-1 chars, adds \0

// C-strings cannot use = for assignment after definition
// city = "Billings";    // WRONG — use strcpy instead

Always leave room for the null terminator — a C-string of N characters needs an array of at least N+1. cin >> stops at whitespace and does no bounds checking. Use cin.getline(array, size) to read strings with spaces and to limit how much is read.

C-String Library Functions

▶

Requires #include <cstring>. These functions work with C-strings (char arrays or pointers to char).

char city[20] = "Missoula";
char location[20] = "Missoula, ";
char state[3] = "MT";
char fname[20] = "Maureen";
char name[20];

// strlen: returns length WITHOUT counting \0
cout << strlen(city);        // 8

// strcat: appends str2 to the END of str1
strcat(location, state);     // location = "Missoula, MT"

// strcpy: copies str2 into str1
strcpy(name, fname);         // name = "Maureen"

// strstr: finds first occurrence of str2 in str1
// returns pointer to match, or NULL if not found
char river[] = "Wabash";
char word[] = "aba";
cout << strstr(river, word); // displays "abash"

// strcmp: compares two C-strings
// returns 0 if equal, negative if str1 < str2, positive if str1 > str2
if (strcmp(name, fname) == 0)
    cout << "Strings are equal\n";

strcat and strcpy do NO bounds checking — if the destination array is too small, they write past the end of the array causing buffer overflow. This is a serious security vulnerability. Always make sure the destination is large enough. The C++ string class is safer — prefer it over C-strings in modern code.

C-String / Numeric Conversion

▶

Requires #include <cstdlib>. Converts between C-strings and numeric types.

int iNum;
long lNum;
double dNum;
char intChar[10];

iNum = atoi("1234");       // "1234" → 1234 (int)
lNum = atol("5678");       // "5678" → 5678 (long)
dNum = atof("35.7");       // "35.7" → 35.7 (double)
itoa(iNum, intChar, 10);   // 1234 → "1234" (base 10)

If the C-string contains non-digit characters, results are undefined — the function may return the result up to the non-digit, or may return 0. itoa does no bounds checking on the destination buffer. For the C++ string class, use stoi(), stol(), stod(), and to_string() instead — they're safer and easier.

// C++11 string class conversions (safer, preferred)
string numStr = "1234";
int n = stoi(numStr);        // string to int
double d = stod("3.14");     // string to double
string s = to_string(42);    // int to string

The C++ string Class

▶

The string class is a special data type that makes string handling dramatically easier and safer than C-strings. Requires #include <string>.

#include <string>

// Define string variables
string firstName, lastName;

// Assign values
firstName = "George";
lastName = "Washington";

// Display
cout << firstName << " " << lastName;

// Input: cin >> stops at whitespace
string word;
cin >> word;

// getline: reads entire line including spaces
string address;
getline(cin, address);

// Compare directly with relational operators
string str1 = "George", str2 = "Georgia";
if (str1 < str2)
    cout << str1 << " comes first\n";

// Access individual characters like an array
for (int i = 0; i < 16; i++)
    cout << phrase[i];

The string class handles memory management automatically — it grows as needed, no fixed size to worry about, no buffer overflows. Relational operators (<, >, ==, !=, etc.) work directly on string objects using lexicographic (alphabetical) comparison. This is far safer and more convenient than C-string equivalents.

string Operators

▶ 
string word1, word2, phrase;
word2 = " Dog";

cin >> word1;          // user enters "Hot Tamale" — word1 gets "Hot"

// + concatenates strings
phrase = word1 + word2;    // "Hot Dog"

// += appends to existing string
phrase += " on a bun";     // "Hot Dog on a bun"

// [] accesses individual characters
cout << phrase[0];         // 'H'
cout << phrase[4];         // 'D'

// = assigns one string to another (unlike C-strings!)
string copy = phrase;      // copy = "Hot Dog on a bun"

string Member Functions

▶

Size & Space

length() / size() — number of chars. empty() — true if empty. capacity() — allocated space. resize(n) — change size.

Modification

append(str) — add to end. insert(pos, str) — insert at position. erase(pos, n) — remove n chars at pos. replace(pos, n, str) — replace. clear() — empty the string.

Substrings & Search

substr(pos, n) — extract n chars at pos. find(str) — position of first match (or string::npos if not found). at(i) — access char at i with bounds checking.

Comparison

compare(str) — like strcmp: 0 if equal, negative if less, positive if greater. Or just use ==, <, > operators directly.

string word1, word2, phrase;
cin >> word1;          // "Hot"
word2.assign(" Dog");

phrase.append(word1);
phrase.append(word2);  // phrase = "Hot Dog"

// append with character count limit
phrase.append(" with mustard relish", 13);
// phrase = "Hot Dog with mustard" (stops at 13 chars)

// insert at position
phrase.insert(8, "on a bun ");
// phrase = "Hot Dog on a bun with mustard"

// find and substr
string sentence = "Hello World";
int pos = sentence.find("World");     // pos = 6
string sub = sentence.substr(6, 5);  // sub = "World"

// length
cout << phrase.length();   // number of characters
